FOOTBALL MATCH SCHEDULER BOT
This project is a Telegram bot that daily fetches upcoming matches from various football leagues including the Champions League, UEFA Europa League, UEFA Conference League, English Premier League, and Turkish Super Lig. The bot compiles a message and sends it to a designated Telegram group at 12:00 PM, providing information about the matches scheduled for that day. Match data is retrieved from "onefootball.com."

FEATURES
Daily match updates for popular football leagues.
Seamless integration with Telegram for easy distribution of match information.
Data sourced from "onefootball.com" for accurate and up-to-date information.
